## About this app

This showcase app exists to exercise every rendering capability of the framework in one place.
Each page focuses on a specific feature set, using real content adapted from the
[official Vue 3 guide](https://vuejs.org/guide/).

The app is intentionally not a reference — see the TCM Primer for a production example.
Its purpose is to surface regressions and confirm that new features work end-to-end before
they are rolled into consumer apps.
