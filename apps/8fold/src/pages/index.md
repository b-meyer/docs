# The Noble Eightfold Path

A short, stock-prose primer used as the **second consumer** of the docs framework
(alongside the TCM Primer). It exercises the framework with nothing but
`framework.config.ts` and these markdown pages — no mermaid, no code blocks.

The Eightfold Path groups eight interdependent factors into three divisions.

## Wisdom (Paññā)

- [Right View](RightView.md) — seeing things as they are.
- [Right Intention](RightIntention.md) — the resolve toward non-harming.

## Ethical conduct (Sīla)

- [Right Speech](RightSpeech.md) — truthful, harmonious, gentle speech.

> [!NOTE]
> This site is a deliberately minimal stub. Its purpose is to prove the
> framework boundary holds for a consumer that ships only config + content.
